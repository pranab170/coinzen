# Coinzen - Decentralized Token Management & Reward System

## Project Description

Coinzen is a comprehensive Solidity-based smart contract project that implements a decentralized token management system with built-in staking and reward mechanisms. The project serves as a complete DeFi solution that allows users to manage tokens, stake them for rewards, and participate in a decentralized ecosystem with transparent and automated reward distribution.

The smart contract combines traditional ERC-20 token functionality with advanced staking features, providing users with the ability to earn passive income through token staking while maintaining full control over their assets. The system is designed with security, transparency, and user experience in mind.

## Project Vision

Our vision is to create a user-friendly, secure, and efficient decentralized token ecosystem that empowers individuals to take control of their digital assets. Coinzen aims to bridge the gap between traditional finance and decentralized finance (DeFi) by providing:

- **Financial Sovereignty**: Users maintain complete control over their tokens without intermediaries
- **Passive Income Generation**: Automated reward system that generates income through staking
- **Transparency**: All transactions and reward calculations are transparent and verifiable on the blockchain
- **Accessibility**: Simple and intuitive interface for users of all technical levels
- **Community-Driven Growth**: A system that rewards long-term participation and community engagement

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- Hardhat or Truffle development environment
- MetaMask or compatible Web3 wallet
- Test ETH for deployment

### Installation
1. Clone the repository
2. Install dependencies: `npm install`
3. Compile contracts: `npx hardhat compile`
4. Run tests: `npx hardhat test`
5. Deploy: `npx hardhat deploy --network <network_name>`

## Core Smart Contract Functions

### Primary Functions:
1. **`transfer()`** - Enhanced token transfer with security validations
2. **`stakeTokens()`** - Stake tokens to earn rewards with time-based calculations
3. **`unstakeTokens()`** - Unstake tokens with automatic reward distribution

### Supporting Functions:
- **`claimRewards()`** - Claim accumulated staking rewards
- **`calculateRewards()`** - View pending reward calculations
- **`getStakingInfo()`** - Get comprehensive staking information
